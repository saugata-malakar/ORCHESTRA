# HackerRank Orchestrate — Support Triage Agent

Starter repository for the **HackerRank Orchestrate** 24-hour hackathon (May 1–2, 2026).

Build a terminal-based AI agent that triages real support tickets across three product ecosystems: **HackerRank**, **Claude**, and **Visa** — using only the support corpus shipped in `data/`.

## Quick Start

```bash
cd code/
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
python main.py            # full run → support_tickets/output.csv
python main.py --sample   # dev run on sample_support_tickets.csv
python main.py --dry-run  # corpus loading only, no API calls
```

## Chat Transcript Logging

Per AGENTS.md §2, the agent logs all conversation turns to:

- macOS / Linux: `$HOME/hackerrank_orchestrate/log.txt`
- Windows: `%USERPROFILE%\hackerrank_orchestrate\log.txt`

This file is gitignored and never committed.

## Architecture Summary

| Layer | Implementation |
|---|---|
| Retrieval | BM25Okapi (offline) + company-affinity boost ×1.35 |
| LLM | `claude-sonnet-4-20250514`, `temperature=0` |
| Escalation | Rule-based pre-screen (safety.py) + LLM judgment |
| Determinism | `random.seed(42)`, pinned deps |
| Grounding | Corpus injected verbatim into prompt; no parametric hallucination |

See `code/README.md` for full architecture details.
