# HackerRank Integrations & Library

## Integrations
HackerRank integrates with major Applicant Tracking Systems (ATS) and HR tools:
- Greenhouse
- Lever
- iCIMS
- Workday
- SmartRecruiters
- Slack
- Microsoft Teams
- Zapier

### Setting Up Integrations
1. Go to Settings > Integrations.
2. Select the integration you want to configure.
3. Follow the setup wizard for authentication and configuration.
4. Map fields between HackerRank and your ATS.

### API Access
- HackerRank provides a REST API for programmatic access.
- API documentation: https://www.hackerrank.com/work/apidocs
- API keys are generated in Settings > API.

## Library
The HackerRank Library contains pre-built questions and assessments:
- Coding questions across 35+ programming languages
- Multiple choice questions
- Project-based assessments
- Role-based test templates (Frontend Developer, Backend Developer, Data Scientist, etc.)

### Using Library Questions
1. Go to Library.
2. Browse or search for questions.
3. Add questions to your test.
4. Customize difficulty and time limits.

### Custom Questions
- Create your own coding, MCQ, or project questions.
- Set custom test cases and scoring criteria.
- Share custom questions within your team.

## Chakra
Chakra is HackerRank's AI-powered assessment generation tool:
- Automatically generates role-specific assessments.
- Uses AI to recommend questions based on job requirements.
- Adapts difficulty based on candidate performance.

## Impersonation Detection
HackerRank offers impersonation detection to verify candidate identity:
- Photo ID verification before test start.
- Periodic face capture during the test.
- AI-powered comparison between ID photo and live captures.
- Reports flagged candidates for reviewer attention.
